NoteNest

I worked on this project in a team of four people. We worked to create a website to convert audio from a guitar to guitar tabliture. There where many difficulties that we had to deal with in the creation of this project. It was very difficult to figure out how to apply fast-fourier tranformations on a website application to be able to decipher the audio information. Furthermore, how a guitar is setup made this a difficult task, as there are up to four different ways to play the same note. To overcome this we had to calcultate the most efficient possible note on the guitar from the previous note that was played.
